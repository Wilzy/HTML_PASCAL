les différents types de variables



Présentation des différentes variables




Les différents types de variables sont divisibles en 5 grandes parties.

  • Les Entiers : sont des nombres qui ne peuvent pas être décimaux (peuvent être positifs ou négatifs)
    se traduit en pascal: INTEGER
  • Les Réels : sont des nombres pouvants utiliser les décimaux (pouvants être négatif ou positif)
    se traduit en pascal: REAL
  • Les caractères : représentent des caractères uniques (comme une touche du clavier)
    se traduit en pascal: CHAR
  • Les chaînes de caractères : sont utilisées pour entrer une chaîne de plusieurs caractères (un texte)
    se traduit en pascal: STRING
  • Les Booléens : sont utilisés pour donner un réponse VRAI ou FAUX ce dit d'une variable suceptible de prendre deux valeurs s'excluant mutuellement, par exemple 0 et 1 (état binaire)
    se traduit en pascal: BOOLEAN
